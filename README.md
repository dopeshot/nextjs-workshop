# Nextjs-workshop

This is a NextJs workshop built as a followup to a React introduction.  

The goal is to provide top level knowledge of the entire eco-system.


## Workshop-Plan

- Jeder circa. 40 Minuten Präsentation
- 80 Minuten Übungen
- 2 ganze Termin
- Jeder kann an einem Termin reden, und übung präsentation kann wechseln
